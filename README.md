# ASM - TM

Cette repository contient tout les Travaux Machine d'introduction a l'assembleur de ma licence d'informatique ainsi qu'un fichier Makefile pour la compilation.

Toutes les consignes associées peuvent se trouver dans le fichier `consignes.md`

Pour compiler un fichier , allez dans un dossier et lancez la commande `make` et executez ensuite le fichier `prog`.
